# simd
SIMD/Vector instructions simulation and test for **amd64**, **arm64**, **ppc64x** and **s390x**.

## Content
- **amd64**  
    - SM3NI
    - SM4NI 
    - SM4 Sbox With AESNI
    - SM4 Sbox With GFNI
    - ZUC Sbox With AESNI
    - ZUC Sbox With GFNI
    - GHASH With CLMUL
    - ZUC With CLMUL
    - Base64
- **arm64**
    - SM3NI
    - SM4NI
    - SM4 Sbox With AESNI
    - ZUC Sbox With AESNI
    - GHASH With CLMUL
    - ZUC With CLMUL
    - Base64
- **ppc64x**
    - XTS
    - SM4 Sbox With AESNI
    - ZUC Sbox With AESNI
    - GHASH With CLMUL
    - ZUC With CLMUL
    - Base64
- **s390x**
    - XTS
    - Base64
